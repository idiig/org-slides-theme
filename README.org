#+title: org-slides-theme
#+options: toc:nil num:nil

ox-html エクスポート用のスライドテーマ。ReadTheDocs スタイルのサイドバーとプレゼンターモードを提供する。

* クイックスタート

org ファイルの先頭に1行追加するだけ：

#+begin_src org
#+SETUPFILE: https://idiig.github.io/org-slides-theme/setup.org
#+end_src

~C-c C-e h h~ でエクスポート後、ブラウザで開く。

* 機能

** ナビゲーション

| キー             | 動作                               |
|------------------+------------------------------------|
| ~→~ / ~←~       | 箇条書き・図・表を1個ずつ表示/非表示 |
| ~↓~ / ~PageDown~ | 次のスライドへ（全表示）           |
| ~↑~ / ~PageUp~   | 前のスライドへ（全表示）           |

** サイドバー

- TOC の自動ハイライト・展開
- スライドタイトルのパンくず表示
- スライドカウンター
- 検索ボックス（Enter で移動、Escape でクリア）

** プレゼンターモード

2つのウィンドウを開き、操作を同期：

- 発表画面 :: ~pres.html~
- プレゼンター画面（セリフ表示）:: ~pres.html?presenter~

どちらで操作しても両画面が連動する（~localStorage~ 経由）。

** スピーカーノート

org ファイルの ~#~ コメントが自動的にノートに変換される：

#+begin_src org
** スライドタイトル

- 箇条書き1
- 箇条書き2
# ここに話す内容を書く。
# 複数行も可能。
#+end_src

プレゼンター画面にのみ表示される。

* 構成ファイル

| ファイル    | 内容                              |
|-------------+-----------------------------------|
| ~setup.org~ | CSS・JS・hooks を一括設定         |
| ~slides.css~ | スタイルシート                   |
| ~slides.js~  | ナビゲーション・同期スクリプト   |
| ~hooks.el~   | org-html エクスポート用 Emacs フック |
